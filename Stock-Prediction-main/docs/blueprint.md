# **App Name**: QuantAxis AI

## Core Features:

- Dashboard Overview: Displays high-level trading analytics, including signal statistics, portfolio return, prediction accuracy, portfolio equity curve, and signal distribution charts.
- AI Signal Engine: A generative AI tool for generating stock trading signals (BUY, SELL, HOLD) with confidence scores, allowing users to select models, stocks, and timeframes.
- Custom Watchlist Management: Allows users to create, modify, and monitor custom stock watchlists with real-time price data and sorting capabilities.
- Strategy Backtesting: Configures and executes trading strategy backtests using historical data, presenting key performance metrics like Sharpe Ratio, Max Drawdown, and Win Rate.
- Portfolio Analytics: Provides an overview of the user's investment portfolio, including total value, daily profit/loss, total return, open positions, and sector allocation breakdown.
- Market Data & Alerts: Displays real-time market data (top gainers/losers/most active) and enables users to configure custom alerts based on various stock conditions.
- Professional User Interface: Includes a customizable sidebar navigation, global stock search, market status indicator, user profile management, and a dark theme toggle for a seamless trading experience.

## Style Guidelines:

- Overall dark fintech theme with a deep, desaturated blue background, ensuring minimal eye strain and focus on data. The background color is #0A0F1C (HSL: 219, 44%, 8%).
- Card and content container backgrounds are a slightly lighter, dark blue-gray at #111827 (HSL: 216, 44%, 11%), providing subtle visual separation from the main background.
- The primary action color is a vibrant blue, #3B82F6 (HSL: 217, 91%, 60%), used for interactive elements, buttons, and primary highlights. It contrasts effectively with the dark scheme.
- Semantic accent colors: Green #22C55E (HSL: 148, 70%, 45%) for 'Buy' signals, Red #EF4444 (HSL: 0, 83%, 60%) for 'Sell' signals, and Yellow #FACC15 (HSL: 48, 97%, 53%) for 'Neutral' or 'Hold' signals.
- The entire application uses the 'Inter' sans-serif font family, offering a modern, clean, and highly readable appearance suitable for both headlines and detailed data tables in a professional environment.
- Utilize a consistent set of clean, minimalist line icons for navigation items and indicators, enhancing the professional and data-focused aesthetic.
- A fixed left sidebar with collapsible functionality and a persistent top navigation bar form the primary application structure, optimized for data-rich desktop displays, with responsive adjustments for tablets and mobile devices.
- Subtle and sophisticated animations are employed using Framer Motion for smooth page transitions, interactive card hover effects, graceful chart loading sequences, and glowing button hover states to enhance user experience without distraction.
