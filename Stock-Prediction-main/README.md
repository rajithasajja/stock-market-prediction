# QuantAxis Pro - Source Code

This directory contains the main source code for the QuantAxis Pro trading terminal.

## 🚀 Getting Started

### 1. Backend (FastAPI)
The backend handles market data retrieval and processing.
- Run: `uvicorn main:app --port 8000` (within `Stock-Prediction-main`)

### 2. Frontend (Next.js)
The frontend provides the user interface and real-time backtesting simulation.
- Run: `npm run dev`

## 🛠️ Key Technologies
- **Next.js 15+**
- **FastAPI**
- **yfinance**
- **Tailwind CSS**

For detailed documentation, see the [README.md](../README.md) in the root directory.
